Consulta de CEP

Explicação API

O que o código faz: Aguarda você clicar no botão "Buscar". Quando você clica, ele pega o CEP que você inseriu.
Com esse CEP, ele acessa a API https://brasilapi.com.br/api/cep/v1/{seu cep aqui} que fornece informações sobre CEPs no Brasil.
Essa API retorna as informações relacionadas ao CEP que você digitou, como cidade, estado, bairro e rua.
O código então mostra essas informações na página para você.
Pra curiosidade você pode pegar esse link https://brasilapi.com.br/api/cep/v1/{seu cep aqui}
com o seu cep e colocar no google, que aparecerá um json da API.
