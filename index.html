Diferença API e Webhook

API (Interface de Programação de Aplicativos): É como uma receita de bolo que diz para um computador como fazer algo específico, como obter informações de um site ou enviar uma mensagem para um amigo.
Você precisa mandar algo, pra ele te retornar uma informação sobre o dado que você enviou
(como foi visto no exemplo da API, aonde você mandou o cep e retornou com as informações da rua, cidade e estado).
Webhook: É como quando você deixa um recado na porta de um amigo dizendo "me avise quando você chegar em casa".
Seu amigo não precisa ligar para você para dizer que chegou em casa, ele apenas te avisa quando chegar. O webhook faz algo parecido, mas entre computadores.
Um sistema diz para outro: "me avise quando algo importante acontecer", e quando acontece, o outro sistema avisa automaticamente, sem precisar ser perguntado.
(como foi feito no exemplo do webhook, quando o contador chega a 10 o site te notifica que o webhook foi ativado).
